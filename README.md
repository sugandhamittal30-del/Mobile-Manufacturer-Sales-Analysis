\# Mobile Sales Data Analysis



\## Project Overview



This project analyzes mobile phone sales data using SQL queries and a Power BI dashboard.

The goal is to extract insights such as top-selling manufacturers, customer spending patterns, and sales trends across different states and years.



\## Files in the Project



\* \*\*Mobile\_sales.sql\*\* – Contains SQL queries used to analyze the sales dataset.

\* \*\*Mobile\_data.sql\*\* – Database structure and data used for the analysis.

\* \*\*Mobile Sales Dashboard.pbix\*\* – Power BI dashboard that visualizes the sales data.



\## Key Analysis Performed



The SQL queries answer several business questions, including:



\* States where customers purchased cellphones since 2005.

\* State in the US with the highest Samsung phone sales.

\* Number of transactions per model by state and zip code.

\* Cheapest cellphone model available.

\* Average price for models from the top 5 manufacturers.

\* Customers with high average spending.

\* Top-performing models across multiple years.

\* Manufacturer sales rankings by year.

\* Customer spending trends and percentage change over time.



\## Tools \& Technologies



\* SQL (for data analysis)

\* Power BI (for data visualization)

\* Relational database structure with fact and dimension tables



\## Project Purpose



This project was created as an \*\*academic data analysis project\*\* to practice SQL querying, data exploration, and dashboard creation.
